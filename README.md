# Buscador de Clima 🌤️

App web que muestra el clima actual de cualquier ciudad del mundo usando la API de OpenWeatherMap.

## Funcionalidades
- Buscar clima por ciudad
- Muestra temperatura, sensación térmica, humedad, descripción y viento
- Manejo de errores para ciudades no encontradas
- Búsqueda con botón o tecla Enter
- Diseño responsive

## Tecnologías usadas
- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## Cómo usarlo
1. Clona el repositorio
2. Consigue tu API key gratis en [openweathermap.org](https://openweathermap.org)
3. En `app.js` reemplaza `'tu_api_key'` con tu key
4. Abre `index.html` en el navegador
